Como funciona?

Qual o objetivo?

Desenvolvi essa pag com o intuito de ser semelhante ao "linktree". Aqui você consegue criar sua própria pag adicionando link's, imagem e descrição. Depois que sua pag é gerada, você poderá compartilha o link dela com amigos.

Como fazer?

Basta preencher o formulário colocando suas informações e links que deseja compartilha.
O formulário contém os seguintes dados que poderam ser preenchidos:

  • Imagem que será exibida na sua pag.
  • Breve Descrição.
  • 3 Buttons que contêm um link.
  • 3 Icons que contêm um link.

Lógica de Desenvolvimento

Quando é preenchido e enviado o formulário, os dados são enviado para o servidor que processa as informação e criar uma nova pag como base no Modelo exemplo. Pós isso o servidor envia os dados via Json pela URL para a pag que foi criada. Essa nova pag processa esses dados e preenche os buttons, icons e img com os dados fornecidos.
